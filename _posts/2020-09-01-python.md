1. 安装homebrew  
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2. 安装pyenv  
Brew update  
brew install pyenv  

3. 安装python  
pyenv install version  

4. 切换版本  
pyenv local 3.7.4
